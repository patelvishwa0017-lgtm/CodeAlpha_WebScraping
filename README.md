# CodeAlpha Web Scraping

## Project Overview

This project demonstrates web scraping using Python by extracting book details from the Books to Scrape website. The scraped data is cleaned, analyzed, and exported as a CSV dataset for further analysis.

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas
* Matplotlib
* Google Colab

## Features

* Extracts book titles, prices, and ratings
* Cleans and processes scraped data
* Performs basic statistical analysis
* Visualizes rating distribution
* Exports data to CSV format

## Output Files

* CodeAlpha_WebScraping.ipynb
* books_dataset.csv
* README.md
